# chat

[![Code Climate](https://codeclimate.com/github/jroliveira/chat/badges/gpa.svg)](https://codeclimate.com/github/jroliveira/chat)

Widget de chat feito em backbone e nodejs.

A aplicação utiliza a lib [localforage](http://mozilla.github.io/localForage/) para registrar as mensagens offline no navegador.

### Installing

* `git clone https://github.com/jroliveira/chat.git`
* `npm install`
* `bower install`

### How to use it

* `node web.js`

##### In action

Se você quer ver rodando clique [aqui](http://jroliveira.net/chat-sample/)
